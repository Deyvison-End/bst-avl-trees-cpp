# BST, AVL Tree e Tree Sort em C++

Projeto desenvolvido durante os estudos de Estruturas de Dados utilizando Árvores Binárias de Busca (BST), Árvores AVL e algoritmos de busca.

## Funcionalidades

- Binary Search Tree (BST)
- AVL Tree
- Inserção de elementos
- Remoção de elementos
- Busca de elementos
- Balanceamento automático
- Tree Sort
- Problema Two Sum

## Conceitos Trabalhados

- Árvores Binárias de Busca
- Árvores AVL
- Recursão
- Balanceamento de árvores
- Rotações simples e duplas
- Percurso In-Order
- Estruturas de Dados
- Complexidade de Algoritmos

## Estrutura do Projeto

### bst.h / bst.cpp

Implementação da árvore binária de busca:

- Inserção
- Busca
- Remoção
- Predecessor
- Sucessor
- Ordenação por percurso In-Order

### avl.h / avl.cpp

Implementação da AVL:

- Balanceamento automático
- Rotação à esquerda
- Rotação à direita
- Rebalanceamento

### tree_sort.cpp

Implementação do algoritmo Tree Sort utilizando AVL.

### two_sum.cpp

Resolução do problema Two Sum utilizando Set.

## Tecnologias

- C++
- STL
- Vector
- Set
- Programação Orientada a Objetos
